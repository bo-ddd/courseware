###  目录结构 

```javascript
 |- node_modules
 |- public
 |- src  //src:source  代表的是源的意思 ，在项目中，代表着 源代码，指的是开发时所写的代码；
	|- assets  //静态资源文件 一般存放的是静态的 css,img,js,lib;
	|- store   // 状态管理工具，(使用场景：用来做复杂组件中相互传参)
	|- router  // 路由
	|- components  //组件；
	|- views  // 视图模块，存放的是所有的html;
 |- tests
 package.json
 README.md
 vue.config.js // 此文件需要手动创建和配置，相当于webpack.config.js，经常用来解决跨域问题
```
